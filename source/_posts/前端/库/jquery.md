## JQuery是什么？

**jQuery 是一个 JavaScript DOM操作的一个库。**  
**jQuery 极大地简化了 JavaScript 编程。**  



## jQuery 库 - 特性

**jQuery** 库包含以下特性：

- HTML 元素选取
- HTML 元素操作
- CSS 操作
- HTML 事件函数
- JavaScript 特效和动画
- HTML DOM 遍历和修改
- AJAX
- Utilities
